# Event Planner Mobile App 

A soccer planner app: view groups and matches (users login, view Event's, join / unjoin Event's)

# Tech Guidelines
-Technologies: React Native + Expo Router 

-Back-end: Event Planer RESTful API, with "Bearer token" auth

-Back-end API source code: `..\event-planner-web\src\app\api`

-Modular design: split the app into meanigful components, to avoid too much code in a single file and reuse repating code.

# Mobile user interface gudelines
- Implement user-friendly UI, stack navigation, responsive layout for tablets and smartphones

- Mobile UI Alerts: ensure all native alerts, confirms and other system dialogs have a fallback for Web (implemented as modal popups)
